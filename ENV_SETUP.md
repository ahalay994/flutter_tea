# Настройка переменных окружения

Для корректной работы приложения необходимо создать файл `.env` в корне проекта со следующими переменными:

```
APP_NAME=Название вашего приложения
API_URL=URL вашего API
SUPABASE_URL=Ваш Supabase URL
SUPABASE_KEY=Ваш Supabase ключ (anon key)
```

**ВАЖНО**: Никогда не добавляйте файл `.env` в репозиторий. Файл уже добавлен в `.gitignore`, чтобы предотвратить случайную отправку секретных данных.

## Настройка названия приложения в iOS

Для корректного отображения названия приложения на iOS (как на иконке приложения, так и в настройках), выполните следующие шаги:

1. Убедитесь, что значение APP_NAME в файле .env заключено в кавычки, если оно содержит пробелы или специальные символы:
   ```
   APP_NAME="Турбо чайна-бабка"
   ```

2. При сборке iOS-версии приложения в Xcode будет запущен скрипт, который автоматически обновит значения в Info.plist:
   - CFBundleDisplayName: отображаемое имя приложения
   - CFBundleName: внутреннее имя приложения

## Получение Supabase ключа

1. Зайдите в ваш проект на https://supabase.com
2. Перейдите в "Project Settings" → "API"
3. Скопируйте "anon key" для использования в качестве SUPABASE_KEY
4. URL проекта будет в поле "Project URL" как SUPABASE_URL

## Безопасность

- Никогда не храните секретные ключи в коде
- Используйте только .env файлы для хранения секретов
- Не делитесь своими ключами с другими
- Регулярно меняйте ключи при необходимости
